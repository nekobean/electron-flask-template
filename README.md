# Electron Flask example

## About

Electron with flask example.

## Setup

```bash
npm install
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## Debug

```bash
npm start
```

## Build

```bash
npm run build
```

Generated package

The generated package will be output to `dist\win-unpacked`.
